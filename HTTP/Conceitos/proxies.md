># Proxies HTTP
- `Representantes`
  - Fica entre o cliente e o servidor
  - Ajudam a fazer o transporte dos dados
  - Roteador é um exemplo de proxie

- `Diversas funções`
  - Cache
  - Filtro (tipo um antivirus, ou controle parental)
  - Load balancing (distribuição da carga)
  - Autenticação
  - Autorização
